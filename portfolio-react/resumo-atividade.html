1. O que é React e como funciona

React é uma biblioteca JavaScript para construir interfaces de usuário. Em vez de escrever um único arquivo HTML gigante, você divide a página em componentes — pedaços reutilizáveis de interface.

Conceito central: No React, a interface é descrita como funções JavaScript que retornam JSX (que parece HTML, mas é JavaScript).

JSX — HTML dentro do JavaScript

JSX é a sintaxe que o React usa. Parece HTML mas tem algumas diferenças:

// HTML normal:
<div class="container">Olá</div>

// JSX (React):
<div className="container">Olá</div>
// Diferença: "class" vira "className" no JSX

3. O que é um Componente

Um componente React é uma função JavaScript que retorna JSX. Cada componente é um arquivo .jsx separado.

// Estrutura básica de um componente:
function NomeDoComponente() {
  return (
    <div>
      <h1>Conteúdo aqui</h1>
    </div>
  )
}

export default NomeDoComponente  // exporta para usar em outros arquivos

Como usar um componente em outro arquivo:

// App.jsx
import Hero from './components/Hero'      // importa
import Skills from './components/Skills'

function App() {
  return (
    <div>
      <Hero />     {/* usa como tag HTML */}
      <Skills />
    </div>
  )
}

4. O que é o Framer Motion

Framer Motion é uma biblioteca de animações para React. Em vez de escrever CSS de animação complexo, você adiciona propriedades diretamente nos elementos.

npm install framer-motion

Como usar:

import { motion } from 'framer-motion'

// Troca qualquer tag HTML por motion.tag:
<motion.div
  initial={{ opacity: 0, y: 50 }}   // estado inicial
  animate={{ opacity: 1, y: 0 }}    // estado final
  transition={{ duration: 0.8 }}     // duração
>
  Conteúdo animado
</motion.div>

Propriedades principais do Framer Motion:

PropriedadeO que faz
initialEstado antes da animação começar
animateEstado final da animação (ao carregar)
whileInViewAnima quando o elemento aparece no scroll
whileHoverAnima quando o mouse passa por cima
transitionControla duração, delay e tipo da animação
viewportConfigura o scroll (ex: once: true = anima só uma vez)

6. Animações Utilizadas

Animação de entrada — Hero (fade + slide)

<motion.section
  initial={{ opacity: 0, y: 50 }}  // começa invisível, 50px abaixo
  animate={{ opacity: 1, y: 0 }}   // termina visível, posição normal
  transition={{ duration: 0.8 }}
>

Scroll Reveal + Stagger — Skills

// O "delay: index * 0.15" faz cada item aparecer depois do anterior
{habilidades.map((item, index) => (
  <motion.li
    initial={{ opacity: 0, x: -30 }}
    whileInView={{ opacity: 1, x: 0 }}
    transition={{ duration: 0.4, delay: index * 0.15 }}
    viewport={{ once: true }}
  >
    {item}
  </motion.li>
))}

Hover Interativo — Cards de Projeto

<motion.div
  whileHover={{ scale: 1.03 }}  // aumenta 3% ao passar o mouse
  transition={{ duration: 0.2 }}
>

Navbar com scroll — useState + useEffect

const [scrolled, setScrolled] = useState(false)

useEffect(() => {
  const handleScroll = () => {
    setScrolled(window.scrollY > 50)  // true se rolou mais de 50px
  }
  window.addEventListener('scroll', handleScroll)
  return () => window.removeEventListener('scroll', handleScroll)
}, [])

// Muda a classe CSS dependendo do estado:
<nav className={scrolled ? 'nav-scrolled' : 'nav-top'}>

7. Conceitos React Aprendidos

useState — guardar valores que mudam

const [scrolled, setScrolled] = useState(false)
// scrolled = valor atual
// setScrolled = função para mudar o valor
// false = valor inicial

useEffect — executar código com efeitos colaterais

useEffect(() => {
  // código que executa quando o componente aparece na tela
  window.addEventListener('scroll', handleScroll)

  return () => {
    // limpeza: remove o listener quando o componente sai da tela
    window.removeEventListener('scroll', handleScroll)
  }
}, []) // [] = executa só uma vez

map() — renderizar listas

const habilidades = ['JavaScript', 'Java', 'Front-end']

// Cria um elemento para cada item da lista:
{habilidades.map((item, index) => (
  <li key={index}>{item}</li>
))}

import/export — compartilhar código entre arquivos

// No arquivo do componente:
export default NomeDoComponente

// Em outro arquivo:
import NomeDoComponente from './components/NomeDoComponente'

className — CSS no React

// HTML usa "class", React usa "className":
<div className="minha-classe"></div>

Navbar.jsx

import { useState, useEffect } from "react"

function Navbar() {
    const [scrolled, setScrolled] = useState(false)

    useEffect(() => {
        const handleScroll = () => {
            setScrolled(window.scrollY > 50)
        }
        window.addEventListener('scroll', handleScroll)
        return () => window.removeEventListener('scroll', handleScroll)
    }, [])

    return (
        <nav className={scrolled ? 'nav-scrolled' : 'nav-top'}>
            <span>Davi Rodrigues</span>
            <ul>
                <li><a href="#inicio">Início</a></li>
                <li><a href="#skills">Skills</a></li>
                <li><a href="#projetos">Projetos</a></li>
                <li><a href="#contato">Contato</a></li>
            </ul>
        </nav>
    )
}

export default Navbar

const habilidades = ['JavaScript', 'Java', 'Front-end', 'Versionamento (GitHub)']

function Skills() {
    return (
        <section id="skills">
            <h2>Conhecimentos</h2>

                        key={index}
                        initial={{ opacity: 0, x: -30 }}
